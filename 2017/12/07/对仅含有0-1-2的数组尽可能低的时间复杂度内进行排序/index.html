对仅含有0,1,2的数组尽可能低的时间复杂度内进行排序

By Liberty_zheng

Published 2017-12-07

Contents

思路：一般来说对排序的复杂度会大于或者O(nlogn)，由于数组中数据分布不清楚如果使用快速排序，其复杂度不确定不满足尽可能低的要求，我们发现数组中只有0,1,2三种数据，可以设三个指针分别指向这三个数字，使0和2分布在1的两边，时间复杂度为O(n)，代码如下：

def  sort(lists):
    left,right,mid=0,len(lists)-1,0
    while mid<=right:
        if lists[mid]==0:
            lists[left],lists[mid]=lists[mid],lists[left]
            mid+=1
            left+=1
        elif lists[mid]==2:
            lists[mid],lists[right]=lists[right],lists[mid]
            right-=1
        else:
            mid+=1
    return lists
