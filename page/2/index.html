找出最长公共连续子串

By Liberty_zheng

Published 2017-10-09

给出两个字符串（可能包含空格），找出其中最长的公共连续子串，输出其长度

思路：算法中常被提及的连续公共子串和连续公共子序列，子序列可以不连续，子串必须连续，采用动态规划的思想，双层循环，当str1[i]==str2[j]时，dp[i][j]=dp[i-1][j-1]+1,否则dp[i][j]=0,代码如下：

str1='_'+raw_input()
str2='_'+raw_input()
dp=[]
for i in range(len(str2)):
   dp.append([0]*len(str1))
for i in range(1,len(str2)):
    for j in range(len(str1)):
        if str2[i]==str1[j]:
            dp[i][j]=dp[i-1][j-1]+1
        else:
            dp[i][j]=0
tmp=[max(x) for x in dp]
print max(tmp)

柱子最大面积

By Liberty_zheng

Published 2017-10-07

给定一组非负整数组成的数组h，代表一组柱状图的高度，其中每个柱子的宽度都为1.在这组柱状图中找到能组成的最大矩形的面积（如图所示）入参h为一个整型数组，代表每个柱子的高度，返回面积的值。

采用分治法，最大矩形面积只可能有三种情况：

  • 高度最小的柱子，面积即为最低高度乘以总长度；
  • 在最低柱子的左面；
  • 在最低柱子右面；

代码如下：

n = input()
heights = map(int, raw_input().split())
def largestarea(a):
    leng = len(a)
    minIndex = a.index(min(a))
    tmp = a[minIndex] * leng  #高低最低时
    if minIndex != 0:
        left = largestarea(a[0:minIndex]) #在最低柱子左面
    else:
        left = 0
    if minIndex != leng - 1:
        right = largestarea(a[minIndex + 1:leng]) #在最低柱子右面
    else:
        right = 0
    return max(left, tmp, right)
print largestarea(heights)

大富翁游戏

By Liberty_zheng

Published 2017-10-07

大富翁游戏，玩家根据骰子的点数决定走的步数，即骰子点数为1时可以走一步，点数为2时可以走两步，点数为n时可以走n步。求玩家走到第n步（n<=骰子最大点数且是方法的唯一入参）时，总共有多少种投骰子的方法。

思路：设f(n)表示到第n步时总共有f(n)种投骰子的方法,那么到第n-1步时总共有f(n-1)种投骰子的方法;那么到第n-2步时总共有f(n-2)种投骰子的方法……
规律:

  • 从第n-1步到第n步，只有一种可能性（投骰子结果为 1 ），所以从第n-1步到第n步共有f(n-1)*1 种投骰子的方法；
  • 从第n-2步到第n步，只有一种可能性（投骰子结果为 2 ），所以从第n-2步到第n步共有f(n-2)*1 种投骰子的方法；
  • 从第n-3步到第n步，只有一种可能性（投骰子结果为 3 ），所以从第n-3步到第n步共有f(n-3)*1 种投骰子的方法；
  • 依次类推:f(n) = f(n-1)+f(n-2)+f(n-3)+….+f(1)+f(0),且f(1)=f(0)=1;
    代码如下：
    import java.util.Scanner;
public class Main{
public static void main(String[] args){
      Scanner scan = new Scanner(System.in);
    int n = scan.nextInt();
    System.out.println(getSum(n));
}
public static int getSum(int n){
        int count=0;
    if(n==1)
    count=1;//f(1) =1
    else if(n==2)
        count=2;//f(2)=2
    else{
        for(int i=1;i<n;i++){
            count+=getSum(n-i);//计算f(n) = f(n-1)+f(n-2)+f(n-3)+....+f(1)
            }
    count=count+1;//再加f(0) = 1
        }
return count;
    }
}

python谷歌专利文件爬取

By Liberty_zheng

Published 2017-10-07

google被称为世界最强大搜索引擎,其反爬虫技术做的也非常好，下面我们就来简单的谈下爬取google专利的一些技巧。

  1. 本篇就用python语言讲述下爬虫的过程，自己写过爬虫程序的人都知道，通常程序有三大主要模块，url管理器，下载器，解析器，我们通常用BeautifulSoup解析器根据自己需求来解析下载好的网页，url管理器也是自己根据需求自己手写，本文章我们重点讲解所用到的下载器。

  2. 通常我们直接访问google，是访问不了的，所以我们要先翻墙，GAE，免费软件和vpn是我们常用的方法，这次我们使用XX-NET代理，至于这个代理是怎么配的，网上有不少的教程，这里我们就不多说了，我们试着访问下google专利网址，谷歌专利，输入一个专利号即可得到专利全文(注意：不翻墙这个地址是访问不了的)
    通过构建url，当专利号为’CN101011964‘时，url构建方法如下：

    import urllib2
    response = urllib2.urlopen(‘https://www.google.com.hk/patents/CN101011964B?cl=zh&dq=CN101011964‘)
    print response.getcode()

    直接抛出401权限错误问题，由于谷歌检测到我们发送的请求并不是通过浏览器，因此尝试下最土的方式试试，每次爬取一个专利文本通过代码启动浏览器并发送请求，如下代码：

    from selenium import webdriver
    browser = webdriver.Chrome()
    browser.get(url)
    html_source = browser.page_source
    return (html_source.encode(‘utf-8’))

注意事项:

  • 该教程适合在winds系统适用；
  • webdriver要去官网去下载,下载后的程序放在chrome浏览器安装目录的application目录下并且在环境变量path中加入路径；
  • 安装selenium，执行pip install selenium即可；
  • 获得一个专利文本后，sleep一段时间再下个专利文本。
  • 最后将获得到的内容使用BeautifulSoup解析即可。
