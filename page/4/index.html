地理散列编码

By Liberty_zheng

Published 2017-10-10

地理散列编码：地理散列常用于将二维的经纬度转换为字符串，分为两步：第一步是经纬度的二进制编码，第二步是base32转码。此题考察纬度的二进制编码：算法对纬度[ -90，90]通过二分法进行无限逼近（取决于所需精度，本题精度为6）。注意，本题进行二分法近近过程中只采用向下取整进来二分，针对二分中间值属于右区间。算法举例如下：针对纬度为80进行二进制编码过程：

  1. 区间[-90，90]进行二分为[-90，0），[0，90]，成为左右区间，可以确定80为右区间，标记为1;
  2. 针对上一步的右区间[0，90]进行二分为[0，45]，[45，90]，可以确定80是右区间，标记为1;
  3. 针对[45，90 ]进行二分为[45，67]，[67,90]，可以确定80为右区间，标记为1;
  4. 针对[67,90]进行二分为[67，78]，[78,90]可以确定80为右区间，标记为1;
  5. 针对[78，90]进行二分为[78，84），[84，90]，可以确定80为左区间，标记 0;
  6. 针对[78，84）进行二分为[78，81），[81，84），可以确定80为左区间，标记为0;

思路：本题思路相对简单，类似于数组的二分查找，根据规则向结果里添加‘1’或‘0’
python代码如下：

from __future__ import division
def f(num):
    values = [-90, 90]
    ret = ''
    for i in range(6):
        mid = int((values[0] + values[1])/2)
        if num >= mid:
            values = [mid, values[1]]
            ret += '1'
        else:
            values = [values[0], mid]
            ret += '0'
    return ret
if __name__ == '__main__':
    while 1:
        try:
            num = int(raw_input())
        except:
            break
        print f(num)

注意:在使用python语言时，需倒入division模块，否则会有少量测试用例无法通过

求满足整数内质数和为该数的个数

By Liberty_zheng

Published 2017-10-10

给定一个正整数，编写程序计算有多少对数数的等于输入的这个正整数，并输出结果。输入值小于1000. 如输入为10，程序应该输出结果为2.（共有两对质数的和为10，分别为（5,5），（3,7））

思路：先求出整数内所有质数的组合，分别判断其和进行计数
python代码如下：

n=input()
from itertools import combinations
def get_zhishu(n):  #    求整数n内所有质数
    tmp=[]
    for i in range(2,n):
        flag=False
        for j in range(2,i):
            if i%j==0:
                flag=True
        if flag==False:
            tmp.append(i)
            flag=False
    return tmp
counts=0
zhishu=get_zhishu(n)
for x in combinations(zhishu,2):
    if x[0]+x[1]==n:
        counts+=1
print counts+1 if n/2 in zhishu else counts

a-y编码

By Liberty_zheng

Published 2017-10-10

假定一种编码的编码范围是a〜y的25个字母，从1位到4位的编码，如果我们把该编码按字典序排序，形成一个数组如下：a，aa，aaa，aaaa，aaab， aaac，… …，b，ba，baa，baaa，baab，baac … …，yyyw，yyyx，yyyy其中索引为0，aa的索引为1，aaa的索引为2，以此类推。编写一个函数，输入是任意一个编码，输出这个编码对应的指数。

思路：由于字母是25个，可以理解为25进制编码，从后向前扫描分别比较每一位字符与字符‘a’的距离，将其距离与转换25进制的数相乘求其和。python 代码如下：

def f(str1, v):
    length = len(str1)
    ret = length - 1
    for i in range(length-1, -1, -1):
        ret += v[i] * int(ord(str1[i]) - ord('a'))  #扫描每个字符与‘a’的距离
    return ret
if __name__ == '__main__':
    v1 = 25*25*25 + 25*25 + 25 + 1
    v2 = 25*25 + 25 + 1
    v3 = 25 + 1
    v4 = 1
    v = [v1, v2, v3, v4]
    while 1:
        try:
            str1 = raw_input()
        except:
            break
        print f(str1, v)

找出最长公共子序列

By Liberty_zheng

Published 2017-10-09

给定一个字符串s，你可以从中删除一些字符，使得剩下的串是一个回文串。如何删除才能使得回文串最长呢？
输出需要删除的字符个数。

思路：其实就是求原字符串和其反串的最大公共子序列,用字符串长度减去其最大公共子序列长度即可,不同于连续公共子串，子序列可以不连续，采用动态规划的思想，双层循环，当str1[i-1]==str2[j-1]时，dp[i][j]=dp[i-1][j-1]+1,否则dp[i][j]=max(dp[i-1][j],dp[i][j-1]),java代码如下:

import java.util.Scanner; 
    public class Main {    
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
            while(in.hasNext()){
            // 相当于求原字符串和其反串的最大公共子序列
                String str1 = in.nextLine();
                String str2 = new StringBuilder(str1).reverse().toString();
                // 动态规划部分
                int[][] c = new int[str1.length()+1][str2.length()+1];
                for(int i=1;i<=str1.length();i++){
                    for(int j=1;j<=str2.length();j++){
                        if(str1.charAt(i-1)==str2.charAt(j-1)){
                            c[i][j]=c[i-1][j-1]+1;
                        }else{
                            c[i][j]=c[i-1][j]>c[i][j-1]?c[i-1][j]:c[i][j-1];
                    }
                }
            }
        System.out.println(str1.length()-c[str1.length()][str2.length()]);
        }

    }
}

找出最长公共连续子串

By Liberty_zheng

Published 2017-10-09

给出两个字符串（可能包含空格），找出其中最长的公共连续子串，输出其长度

思路：算法中常被提及的连续公共子串和连续公共子序列，子序列可以不连续，子串必须连续，采用动态规划的思想，双层循环，当str1[i]==str2[j]时，dp[i][j]=dp[i-1][j-1]+1,否则dp[i][j]=0,代码如下：

str1='_'+raw_input()
str2='_'+raw_input()
dp=[]
for i in range(len(str2)):
   dp.append([0]*len(str1))
for i in range(1,len(str2)):
    for j in range(len(str1)):
        if str2[i]==str1[j]:
            dp[i][j]=dp[i-1][j-1]+1
        else:
            dp[i][j]=0
tmp=[max(x) for x in dp]
print max(tmp)

柱子最大面积

By Liberty_zheng

Published 2017-10-07

给定一组非负整数组成的数组h，代表一组柱状图的高度，其中每个柱子的宽度都为1.在这组柱状图中找到能组成的最大矩形的面积（如图所示）入参h为一个整型数组，代表每个柱子的高度，返回面积的值。

采用分治法，最大矩形面积只可能有三种情况：

  • 高度最小的柱子，面积即为最低高度乘以总长度；
  • 在最低柱子的左面；
  • 在最低柱子右面；

代码如下：

n = input()
heights = map(int, raw_input().split())
def largestarea(a):
    leng = len(a)
    minIndex = a.index(min(a))
    tmp = a[minIndex] * leng  #高低最低时
    if minIndex != 0:
        left = largestarea(a[0:minIndex]) #在最低柱子左面
    else:
        left = 0
    if minIndex != leng - 1:
        right = largestarea(a[minIndex + 1:leng]) #在最低柱子右面
    else:
        right = 0
    return max(left, tmp, right)
print largestarea(heights)

大富翁游戏

By Liberty_zheng

Published 2017-10-07

大富翁游戏，玩家根据骰子的点数决定走的步数，即骰子点数为1时可以走一步，点数为2时可以走两步，点数为n时可以走n步。求玩家走到第n步（n<=骰子最大点数且是方法的唯一入参）时，总共有多少种投骰子的方法。

思路：设f(n)表示到第n步时总共有f(n)种投骰子的方法,那么到第n-1步时总共有f(n-1)种投骰子的方法;那么到第n-2步时总共有f(n-2)种投骰子的方法……
规律:

  • 从第n-1步到第n步，只有一种可能性（投骰子结果为 1 ），所以从第n-1步到第n步共有f(n-1)*1 种投骰子的方法；
  • 从第n-2步到第n步，只有一种可能性（投骰子结果为 2 ），所以从第n-2步到第n步共有f(n-2)*1 种投骰子的方法；
  • 从第n-3步到第n步，只有一种可能性（投骰子结果为 3 ），所以从第n-3步到第n步共有f(n-3)*1 种投骰子的方法；
  • 依次类推:f(n) = f(n-1)+f(n-2)+f(n-3)+….+f(1)+f(0),且f(1)=f(0)=1;
    代码如下：
    import java.util.Scanner;
public class Main{
public static void main(String[] args){
      Scanner scan = new Scanner(System.in);
    int n = scan.nextInt();
    System.out.println(getSum(n));
}
public static int getSum(int n){
        int count=0;
    if(n==1)
    count=1;//f(1) =1
    else if(n==2)
        count=2;//f(2)=2
    else{
        for(int i=1;i<n;i++){
            count+=getSum(n-i);//计算f(n) = f(n-1)+f(n-2)+f(n-3)+....+f(1)
            }
    count=count+1;//再加f(0) = 1
        }
return count;
    }
}

python谷歌专利文件爬取

By Liberty_zheng

Published 2017-10-07

google被称为世界最强大搜索引擎,其反爬虫技术做的也非常好，下面我们就来简单的谈下爬取google专利的一些技巧。

  1. 本篇就用python语言讲述下爬虫的过程，自己写过爬虫程序的人都知道，通常程序有三大主要模块，url管理器，下载器，解析器，我们通常用BeautifulSoup解析器根据自己需求来解析下载好的网页，url管理器也是自己根据需求自己手写，本文章我们重点讲解所用到的下载器。
  2. 通常我们直接访问google，是访问不了的，所以我们要先翻墙，GAE，免费软件和vpn是我们常用的方法，这次我们使用XX-NET代理，至于这个代理是怎么配的，网上有不少的教程，这里我们就不多说了，我们试着访问下google专利网址，谷歌专利，输入一个专利号即可得到专利全文(注意：不翻墙这个地址是访问不了的)
    通过构建url，当专利号为’CN101011964‘时，url构建方法如下：
import urllib2 
response = urllib2.urlopen('https://www.google.com.hk/patents/CN101011964B?cl=zh&dq=CN101011964') 
print response.getcode()

直接抛出401权限错误问题，由于谷歌检测到我们发送的请求并不是通过浏览器，因此尝试下最土的方式试试，每次爬取一个专利文本通过代码启动浏览器并发送请求，如下代码：

from selenium import webdriver
browser = webdriver.Chrome()
browser.get(url)
html_source = browser.page_source
return (html_source.encode('utf-8'))

注意事项:

  • 该教程适合在winds系统适用；
  • webdriver要去官网去下载,下载后的程序放在chrome浏览器安装目录的application目录下并且在环境变量path中加入路径；
  • 安装selenium，执行pip install selenium即可；
  • 获得一个专利文本后，sleep一段时间再下个专利文本。
  • 最后将获得到的内容使用BeautifulSoup解析即可。
